### Базовый класс - SportsmenInGym 
#### Класс из ЛР1 , немного изменен 
* Добавлен метод registration для общего интерфейса
### Дочерние классы : Trainer,EduSportsmen
#### 1. Trainer - Тренер , работник зала 
Наследует от базового :
* Имя 
* Состояние (нахождение в зале)
Новое:
* Возраст 
* Опыт 
* Стоимость 1ч тренировки
* Количество мест для записи 
### 2. EduSportsmen - Спортсмен , коорый занимаетсяс тренером 
Наследует :
* Имя 
* Состояние 
* Уровень абонимента 
Новое:
* Количество будующих , опалченых тренировок с тренером 
### Различия : Разные атрибуты , функция  registration работает по разному (для тренера уменшает сободные места на 1 , для спортсмена увеличивает количество тренировок (на любое число)) , есть и другие методы , для тренера есть функция добвления свободных слотов , для спортсмена функция заканчивающая тренировку и уменьшающая количество у него тренировок.
